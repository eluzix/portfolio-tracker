## Codebase Patterns

- **Data units**: All monetary values in `types.AnalyzedPortfolio` are stored in cents (int32/int64), require division by 100 for display
- **Async pattern**: Use `tea.Cmd` pattern for async operations that return messages (InsightsLoadedMsg, InsightsErrorMsg)
- **LLM integration**: OpenAI client initialization can fail if OPENAI_KEY env var is missing - handle gracefully in modal
- **Modal context**: All modal-related types (ModalType, messages) centralized in `tui/messages.go`
- **Insights modal**: Uses `charmbracelet/bubbles/viewport` for scrollable content, styled with custom InsightsStyles
